# 聊聊 ChatSpace - 在线社交聊天平台

## 快速启动

### 方式一：一键启动（推荐）
双击 `启动聊天服务器.bat` 文件，服务器会自动启动，打开浏览器访问显示的地址即可。

### 方式二：手动启动
1. 打开命令行（CMD 或 PowerShell）
2. 进入项目目录
3. 运行：`npm install` （首次需要安装依赖）
4. 运行：`node server.js`
5. 打开浏览器访问 http://localhost:3000

## 如何让不同设备的人一起聊天

### 方法A：局域网（同一WiFi）—— 最简单

服务器启动后会显示两个地址：
- **本机访问**: http://localhost:3000 （自己电脑浏览器）
- **局域网访问**: http://你的IP:3000 （同一WiFi下的手机、平板、其他电脑）

只要你的电脑和其他设备在同一个 WiFi/局域网下，其他设备在浏览器输入局域网地址就能访问和聊天！

**注意**：如果你的电脑有防火墙，可能需要放行 3000 端口的入站连接。

### 方法B：互联网（任何地方都能访问）—— 推荐 Railway 部署

如果你想让**任何地方的人**通过互联网访问，最简单的方法是部署到 Railway（免费云平台）：

#### 步骤：

1. **注册 GitHub 账号**（如果没有）→ https://github.com/signup
2. **注册 Railway 账号**（用 GitHub 登录）→ https://railway.app
3. **把项目上传到 GitHub**：
   - 在 GitHub 创建一个新仓库（New Repository）
   - 名称随意，比如 `liaoliao-chat`
   - 选择 Private（私有）或 Public（公开）
   - 在项目目录运行以下命令：
   ```
   git init
   git add .
   git commit -m "聊聊聊天网站"
   git remote add origin https://github.com/你的用户名/liaoliao-chat.git
   git push -u origin main
   ```
4. **在 Railway 部署**：
   - 登录 Railway → 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 选择你刚上传的 `liaoliao-chat` 仓库
   - Railway 会自动检测 Node.js 项目并部署
   - 等几分钟后，Railway 会给你一个永久链接，比如 `liaoliao-chat.up.railway.app`
5. **分享链接**：把这个链接发给任何人，他们就能从任何地方访问和聊天了！

#### Railway 免费额度：
- 每月 $5 免费额度，够个人使用
- 每月 500 小时运行时间（约 21 天）
- 如果超了会暂停，下月自动恢复

### 方法C：ngrok 临时隧道（适合临时测试）

1. 下载 ngrok：https://ngrok.com/download
2. 注册 ngrok 账号（免费）
3. 运行：`ngrok http 3000`
4. ngrok 会给你一个临时链接（如 `https://xxxx.ngrok-free.app`）
5. 把链接发给朋友即可访问
6. **缺点**：每次重启 ngrok 链接会变

- 封禁/删除用户
- 删除违规动态

## 功能介绍

- 📝 **注册/登录** - 任何人都可以注册账号
- 💬 **私聊** - 和好友一对一实时聊天，支持文字和图片
- 👥 **群聊** - 加入/创建群组，多人实时聊天
- 🌟 **动态广场** - 发动态、点赞、评论，支持多张图片
- 🔍 **发现** - 推荐用户、热门群聊，一键加好友
- 🛡️ **管理后台** - 管理员专属，监控和管理平台

## 技术架构

- 后端：Node.js + Express + Socket.IO
- 前端：原生 HTML/CSS/JavaScript
- 实时通信：WebSocket (Socket.IO)
- 数据存储：JSON 文件（轻量级，无需数据库）
- 图片上传：multer + 本地存储
